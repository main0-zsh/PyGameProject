# PyGame Dino Clone

Клон знаменитой игры про динозаврика из Google Chrome, написанный на Python с использованием библиотеки Pygame.

## 🎮 Особенности проекта
В репозитории представлены две стадии разработки игры:

1. **Simple Version (Квадраты):** Упрощенная версия, где вместо графики используются геометрические фигуры. Фокус сделан на логике прыжков, обработке столкновений и генерации препятствий.
2. **Full Graphics Version (Original):** Полноценная игра с оригинальными спрайтами динозавра, кактусов и анимацией. Включает в себя систему подсчета очков и ускорение игры со временем.

## 🛠 Технологии
* **Python 3**
* **Pygame** (библиотека для разработки игр)

## 🚀 Как запустить

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/ваш-логин/PyGameProject.git](https://github.com/ваш-логин/PyGameProject.git)
   cd PyGameProject
Установите Pygame:
Убедитесь, что у вас активно виртуальное окружение, и выполните:

Bash
./.venv/bin/python3 -m pip install pygame
Запуск игры:

Для запуска версии с графикой:

Bash
./.venv/bin/python3 main.py
Для запуска упрощенной версии (если она в другом файле, например simple_dino.py):

Bash
./.venv/bin/python3 simple_dino.py
🕹 Управление
Пробел / Стрелка вверх: Прыжок

ESC: Выход из игры
